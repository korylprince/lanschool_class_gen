Help

You can search for students by their username or their name. When searching by name you can use the * as a wild card. For instance Jo* Smith would match both John Smith and Johnny Smith.

If your search returns more than one student, a list will be returned that will allow you to select the student you want.

Click on individual students to remove them from the list or click the Delete All button to remove all students at once.

Once you have the students you want in your list, choose a file name and click the Download button to download your Class List.

If you have a file you would like to modify, click the browse button to select it and then click the Import button to import the students to your current list. When importing a Class List, the students you have currently selected and the students in the file you import will be merged.

Please note that only .lsc files can be uploaded, and only Class Lists based on username will work.
